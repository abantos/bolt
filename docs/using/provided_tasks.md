# Bolt Provided Tasks

Bolt provides the implementation of a few common tasks that most
[Python](https://www.python.org) projects should be able to leverage their
functionality. These tasks are registered when Bolt is executed, and users only
need to configure and invoke them in their `boltfile.py`.
